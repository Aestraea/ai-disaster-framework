An AI-Driven Multi-Label Disaster Report Classification Using NLP and Machine Learning with Automated Preprocessing

MODEL FILES
The RoBERTa models are not included in this repository due to GitHub file size limits.

Download from Google Drive and place in the /backend folder.
roberta_model_disaster  (RoBERTa Tagalog - Disaster Classification)
Link: https://drive.google.com/drive/folders/1OL8OCgS0Rfym7DyCcKMjt0k48oKO0CIl?usp=sharing

roberta_model_location  (RoBERTa NER - Location Extraction)
Link: https://drive.google.com/drive/folders/1-_h_STrYJPzfCyoNMgypWD1p13_GQ2XW?usp=sharing

SETUP
| Install dependencies
| cd backend
| pip install -r requirements.txt
| Run the backend
| python app.py
| Open the website
    Link: http://127.0.0.1:5000

TECH STACK
| Frontend  : HTML, CSS, JavaScript
| Backend   : Python, Flask
| Database  : Firebase Firestore
| ML        : scikit-learn, HuggingFace Transformers, PyTorch

MODELS
| Disaster Classification  : Logistic Regression, SVM, RoBERTa Tagalog
| Population Group         : Logistic Regression, SVM, RoBERTa (placeholder)
| Location Extraction      : RoBERTa NER

LABELS
| Disaster Type    : Flood, Fire, Earthquake, Landslide
| Population Group : Large, Medium, Small, Unknown
| Location         : Municipalities across Bicol Region
